SEARCH

/search_persons/Ole returns all persons with first or lastname containing ole

POST

Methods for CREATE opperations

All POST requeste are expected as JSON


/players Create player with these parameters: {
"person_id": int, "team_id": int, "normal_position": String, "number": int, "activity": true/false
}

/associations Create associations with these parameters: {
"name": String, "description": String, "activity": true/false
}

/coaches Create coaches with these parameters: {
"person_id": int, "activity": true/false
}

/owners Create owners with these parameters: {
"person_id": int, "activity": true/false
}

/persons Create persons with these parameters: {
"address_id": int, "first_name": String, "last_name": String, "date_of_birth": Date, "activity": true/false
}

/contacts Create contacts with these parameters: {
"person_id": int, "contact_type": String, "contact_detail": String, "activity": true/false
}

/match_positions Create match positions with these parameters: {
"player_id": int, "match_id": int, "position": String, "activity": true/false
}

/results Create results with these parameters: {
"match_id": int, "team_id": int, "result": String, "score": int, "activity": true/false
}

/teams Create teams with these parameters: {
"association_id": int, "coach_id": int, "location_id": int, "name": String, "activity": true/false
}

/addresses Create addresses with these parameters: {
"address_line_1": String, "address_line_2": String(can be null), "address_line_3": String(can be null), "postal_code": String, "city": String, "country": String, "activity": true/false
}

/seasons Create seasons with these parameters: {
"start_date": date, "end_date": date, "name": String, "description": String, "activity": true/flalse
}

/locations Create locations with these parameters: {
"address_id": int, "name": String, "description": String, "activity": true/false
}

/goal_types Create goal types with these parameters: {
"type": String, "activity": true/false
}

/match_goals Create match goals with these parameters: {
"player_id": int, "goal_type_id": int, "description": String, "activity": true/false, "match_Id": int
}

/matches Create matches with these parameters: {
"match_date": date, "home_team_id": int, "away_team_id": int, "season_id": int, "location_id": int, "activity": true/false
}
